## Example Usage - VPC Access Connector


```typescript
import * as pulumi from "@pulumi/pulumi";
import * as gcp from "@pulumi/gcp";

const connector = new gcp.vpcaccess.Connector("connector", {
    ipCidrRange: "10.8.0.0/28",
    network: "default",
    region: "us-central1",
});
```
